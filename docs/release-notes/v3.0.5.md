# Release Notes - v3.0.5

## ✨ What's New
- **Modern Tabbed UI**: Completely redesigned interface with a clean, modern look. Features are now organized into "Customization" and "Productivity" tabs for better accessibility.
- **Enhanced Thumbnail Control**: Precision control over how YouTube thumbnails appear—choose between Show Normally, Blur (with unblur on hover), or Hide Completely.
- **Productivity Suite**:
  - **Time Reminder**: Set smart notifications to alert you after a period of continuous watching.
- **Ad Blocker Popup Remover**: Automatically detects and dismisses YouTube's ad blocker detection popups.
- **Block Video Previews**: Option to stop annoying auto-playing video previews when hovering over thumbnails.

## 🛠️ Bug Fixes
- **Mozilla Compatibility**: Resolved security warnings and manifest errors specifically for Firefox.
- **Visual Improvements**: 
  - Fixed an issue where channel avatars were incorrectly blurred.
  - Resolved a bug that caused search results to sometimes be hidden.
  - Fixed "empty video shell" issues during playback.
- **Icon Stability**: Fixed missing or corrupted icon issues in the browser toolbar.
- **Performance**: Optimized the "Unblur on Hover" logic for smoother transitions.

## 📦 Technical Updates
- **Webpack Migration**: Fully migrated to Webpack for faster build times and cross-platform optimization.
- **Hot Reload**: Added development support for hot reloading to speed up feature delivery.
- **SEO & Docs**: Enhanced landing page and documentation for better discoverability.
